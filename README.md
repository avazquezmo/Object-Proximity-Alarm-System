# Object-Proximity-Alarm-System - v.1.0
# By Axel Vazquez
Personal Project:

Description:
  This project is a sensor that contains a visual and auditive alarm that displays when an object is at a certain distance

Materials:
  1 Arduino uno rev3
 
  1 Ultrasonic sensor
 
  1 Buzzer
 
  2 LED (red and green)
 
  1 Breadoard
 
  11 Jumper wires
